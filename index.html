The \Illuminate\Support\Collection class has built-in support for JSON serialization.

This patch adds XML serialization to the following classes:

  • \Illuminate\Support\Collection (toXml() method)
  • \Illuminate\Support\MessageBag (toXml() method)
  • Illuminate\Database\Eloquent\Model (toXml() method)
  • Illuminate\Support\Facades\Response (xml() method)

No external dependencies This code utilizes XMLWriter class which is bundled with PHP since 5.1.x
